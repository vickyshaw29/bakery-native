
# File System

The app is made using react-native and firebase


## Features
- Authenticate using firebase
- Register , Login
- Fullscreen mode
- Cross platform

  

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/vickyshaw29/bakery-native.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
